# Contact management Server
* Clone this repo
```
git clone https://github.com/ProdipKumarShil/neutron-frontend
```
* Enter neutron-frontend
```
cd neutron-frontend
```
* Install node packages
```
npm install
```
* Run this application
```
npm run dev
```
* App will run in here. Go to browser and hit this url
```
http://localhost:5173/
```
